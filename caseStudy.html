CouchBuddies Web App

Helping friends organize movie nights with an intuitive, interactive platform.

Background

Friends often struggle to decide what to watch, forget interesting movie titles, or lose track of movie ideas. CouchBuddies centralizes movie storage and planning to make movie nights fun and effortless.

Issues / Problems

  • Difficulty remembering or sharing movie ideas among friends.
  • No centralized platform for movie night planning.
  • Lack of a quick way to provide feedback on movie suggestions.
  • No visual representation of movies for better selection.

Process / Actions / Decisions

Identified core user needs, then chose React for an interactive front-end and Supabase for secure back-end and authentication. Implemented hover-to-expand movie cards, thumbs up/down ratings, and planned API integration for poster fetching.

Work

  • React components: Dashboard, MovieCard, AddMovieForm
  • Supabase back-end for authentication, database, and API endpoints
  • Interactive features: hover expansion, thumbs up/down, edit/delete functionality
  • Planning API integration for poster fetching
Code Highlight: 
exports.getMoviePoster = async (req, res) => {
  const { title } = req.query;

  if (!title) return res.status(400).json({ posterUrl: '/placeholder-poster.png' });

  try {
    const apiKey = process.env.OMDB_API_KEY;
    const response = await fetch(`http://www.omdbapi.com/?t=${encodeURIComponent(title)}&apikey=${apiKey}`);
    const data = await response.json();

    if (data.Response === "True" && data.Poster && data.Poster !== "N/A") {
      return res.json({ posterUrl: data.Poster });
    }

    return res.json({ posterUrl: '/placeholder-poster.png' });
  } catch (error) {
    console.error('Error fetching poster:', error);
    return res.status(500).json({ posterUrl: '/placeholder-poster.png' });
  }
};

Result

CouchBuddies reduced indecision and planning time for movie nights while improving engagement. Fully responsive, the app works on desktops, tablets, and mobile devices. API integration enhances visual appeal and user experience.
